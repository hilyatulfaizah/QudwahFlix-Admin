# QudwahFlix Admin Dashboard 🛠️📊

A web-based admin dashboard for managing the QudwahFlix platform. Built to handle users, courses, products, orders, and payments with a clean interface and real-time data updates.

---

## 🚀 Overview
QudwahFlix Admin Dashboard is designed to give administrators full control over the platform’s content and operations.

This project demonstrates practical experience in:
- Admin panel development  
- Data management systems  
- CRUD operations  
- API integration  
- Dashboard UI/UX design  

---

## ✨ Key Features
- 👥 User management (view, edit, delete users)  
- 🎥 Course management (add, update, remove courses)  
- 🛍️ Product management  
- 📦 Order tracking and management  
- 💳 Payment monitoring (subscription & installment)  
- 📊 Dashboard analytics overview  
- 🔐 Secure admin authentication  
- ⚡ Real-time database updates  

---

## 🛠️ Tech Stack
